== Description ==

The Openbookings Calendar Plugin displays booking availability of an object defined in your openbookings.org system.

A full calendar year is displayed, with booked dates highlighted starting from now.

You need to have installed openbookings.org on the same database as Wordpress
(see http://openbookings.org Â© for details

== Installation ==

It's very simple :

1. Upload the `openbookings` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[[openbookings gite="1"]]` to display the calendar of object numer 1 in your openbookings.org
